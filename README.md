# Web Scraping with Python

This repository contains my hands-on practice and learning of **Web Scraping using Python**. The notebook covers the basics of extracting data from websites using the `requests` library and `BeautifulSoup`.

The main goal of this project is to understand how web scraping works and how to collect useful information from HTML pages for further analysis or data projects.

---

## Topics Covered

- Sending HTTP requests
- Parsing HTML with BeautifulSoup
- Working with HTML Tags
- Navigating HTML elements
- Extracting text from web pages
- Finding elements by Class
- Finding elements by ID
- Extracting hyperlinks
- Extracting image URLs
- Working with comments in HTML
- Understanding Navigable Strings

---

## Technologies Used

- Python
- Jupyter Notebook
- Requests
- BeautifulSoup (bs4)

## What I Learned

While working on this project, I learned:
